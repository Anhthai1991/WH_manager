# KPI Chính – Warehouse (Rawlplug Việt Nam)

1. Inventory Accuracy (Độ chính xác tồn kho)  
2. Inventory Turnover (Vòng quay tồn kho)  
3. SKU-to-Order Fulfillment Accuracy (Độ chính xác xuất theo SKU)  
4. Order Accuracy / Perfect Order Rate (Đơn hàng đúng – đủ – không lỗi)  
5. On-time Shipment (Tỷ lệ giao hàng đúng hạn)  
6. Warehouse Throughput (Khối lượng xử lý kho: inbound/outbound)  
7. Picking / Packing Productivity (Năng suất xử lý đơn)  
8. Space Utilization (Sử dụng không gian kho)  
9. Shrinkage / Loss Rate (Tỷ lệ hao hụt, hư hỏng, mất mát)  
10. Cost per Order / Cost per Unit (Chi phí xử lý mỗi đơn / mỗi sản phẩm)  

# KPI Chi Tiết – Kho / Logistics – Rawlplug Việt Nam

| Nhóm KPI | KPI | Công thức / cách đo | Target gợi ý | Ghi chú |
|---------|------|---------------------|----------------|---------|
| **Tồn kho & chính xác** | Inventory Accuracy | (Tồn theo hệ thống − tồn thực tế) / tồn hệ thống | ≥ 98–99% | Quan trọng vì nhiều SKU, nhiều mã nhỏ dễ nhầm |
| | Inventory Turnover | Doanh số bán / tồn kho trung bình | ≥ 4–6 vòng/năm (tùy dòng fasteners) | Giúp giảm tồn kho chết, vốn tồn cao |
| **Xuất hàng & chất lượng đơn** | SKU-to-Order Accuracy | Số SKU đúng mã / tổng SKU xuất | ≥ 99% | Tránh nhầm mã, thiếu/sai mã do đa dạng sản phẩm |
| | Order Accuracy / Perfect Order Rate | Đơn giao đúng + đủ + đúng hàng | ≥ 99% | Quan trọng với khách hàng, độ tin cậy cao |
| | On-time Shipment | Đơn giao đúng hẹn / tổng đơn | ≥ 95–98% | Phản ánh hiệu quả kho + logistics + kế hoạch |
| **Hiệu suất kho & xử lý** | Warehouse Throughput | Số lượng pallet / kg / đơn hàng xử lý / ngày | ≥ baseline + cải tiến 5–10%/năm | Đo năng lực kho — inbound + outbound + chuyển |
| | Picking / Packing Productivity | Lines (đơn, kiện, SKU…) / giờ / người | Ví dụ: 120–200 lines/giờ/người — tùy SKU & layout | Đánh giá nhân công & hiệu quả vận hành |
| **Không gian & chi phí vận hành** | Space Utilization | % diện tích/kệ/hệ thống kho được dùng | 70–85% | Giữ buffer → linh hoạt với nhập hàng, SKU mới |
| | Shrinkage / Loss Rate | Số lượng/giá trị hàng hư – mất / tổng hàng | ≤ 0.2–0.5% | Quan trọng với hàng nhỏ, dễ thất lạc, sai mã |
| | Cost per Order / Cost per Unit | Tổng chi phí kho + nhân công + xử lý / đơn hoặc / đơn vị sản phẩm | Giảm dần 5–10%/năm | Giúp kiểm soát chi phí, tối ưu vốn lưu động |
