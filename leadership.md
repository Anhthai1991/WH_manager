# KPI Leadership – Warehouse Manager

| Nhóm | KPI | Target gợi ý | Ý nghĩa |
|------|-----|--------------|---------|
| Phát triển con người | Training Completion Rate | 100% | Tất cả nhân viên được đào tạo SOP, an toàn |
| | Multi-skill Ratio | ≥ 50–70% | Giúp linh hoạt điều phối trong cao điểm |
| | Productivity per head | Cải thiện 5–10%/năm | Đo hiệu quả đào tạo & sắp xếp nhân lực |
| Văn hoá & gắn kết | Employee Turnover Rate | ≤ 10–12%/năm | Giảm nghỉ việc, tăng ổn định đội ngũ |
| | Absenteeism Rate | ≤ 2–3% | Đo tinh thần – trách nhiệm |
| Team performance | On-time task completion | ≥ 95% | Năng lực thực hiện đúng cam kết |
| | 5S / Audit Score | ≥ 90% | Thể hiện tính kỷ luật & môi trường làm việc |
